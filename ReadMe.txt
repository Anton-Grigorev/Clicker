Кликер
Будет полезен в браузерных играх и специальных сервисах, где необходимо нудно и долго кликать, чтобы получить бонусы. 
Сферу применения можно определить самому.
Присутствует функионал "быстрых клавиш":
- запуск кликера: Alt + Q
- выбор времени задержки клика: Alt + I 
- остановка кликера принудительно: Alt + S 
Необходимо импортировать следующие библиотеки:
import keyboard                                                     
import mouse                                                        
import time  
Запустить можно в любой среде разработки.

Clicker
It will be useful in browser games and special services where you need to click tediously and for a long time to get bonuses. 
The scope of application can define yourself.
There is a functionality of "shortcuts":
- start clicker: Alt + Q
- click delay time selection: Alt + I
- stop the clicker forcibly: Alt + S
You need to import the following libraries:
import keyboard
import mouse
import time
You can run in any development environment.
